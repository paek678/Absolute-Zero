## 해결된 질문 — 21개

### ✅ Q1. 핫팩 미니게임 — 시간과 횟수 (2026-07-17)

**5초 동안 10회 연타.** 수치는 "아이템 목록" 스프레드시트가 기준.

### ✅ Q2. 따뜻한 차 — 사용 횟수 (2026-07-17)

**1회 사용 후 소멸.** 특수 아이템(스마트폰) 제외 모든 소모 아이템은 1회 사용 후 소멸.
> ⚠ 코드 영향: 현재 차 MaxUses=2 (임시값) → **1로 수정 필요**

### ✅ Q3. 고양이 — 사용 횟수 (2026-07-17)

**1회.** 모든 소모 아이템은 1회 사용 후 소멸. (현재 코드와 일치)

### ✅ Q4. 삼계탕 "+3, -7" 효과 방향

즉시 상대 온도 +3° (올림/회복) → 다음 턴 상대 온도 -7° (내림/공격). 둘 다 상대에게 적용.

### ✅ Q5. 탄산음료 "-5, +15" 효과 방향

즉시 내 온도 -5° (자해 비용) → 다음 턴 내 온도 +15° (이득). 둘 다 자신에게 적용.

### ✅ Q7. 드롭 확률 합계 102% (2026-07-17)

**"아이템 목록" 스프레드시트에서 조정 예정.** 확정 전까지 현재 가중치 풀(/102) 유지, 확정 시 수치만 교체.

### ✅ Q9. 타로카드 "추가 사용" 타이밍 (2026-07-17)

1. **준비 시간 내 즉시 사용 가능** (타로카드만 해당하는 즉발형)
2. **타로카드 사용은 "아이템 사용"으로 카운트하지 않는다** — 사용 후 아이템을 한 번 더 선택 가능
   → 타로로 상대 선택을 알아낸 뒤 카운터 아이템으로 대응하는 심리전 설계

### ✅ Q10. 미니게임 중 상대방 화면 (2026-07-17)

**아무것도 안 보임** — 상대는 자기 아이템 선택 진행(완료 시 대기 상태).
- 준비 시간 중 상대에게 전달되는 모션은 **"대기" / "준비 끝 버튼 누름" 두 가지뿐**
- 예외: **속마음 타로카드만 사용 즉시 상대방에게 시각적으로 전달됨**

### ✅ Q11. 미니게임 판정 권한

클라이언트에서 판정 → 성공/실패 결과만 ServerRpc로 전송.
서버 타이머 만료 시 → 강제 실패 처리 (서버가 마스터).

### ✅ Q12. 미니게임 실패 시 아이템 소모

소모 안 됨. 실패해도 아이템 유지, 다시 선택하여 재도전 가능.

### ✅ Q13. 공격턴 아이템 사용 연출

전용 애니메이션/이펙트 예정. 아트 확정 후 결정.

### ✅ Q14. 선풍기 월드 표시

StayItem 위치에 오브젝트 스폰만. 연출은 추후 추가.

### ✅ Q15. 상대방 아이템 보유 목록

EnemyItem 위치에 서버 데이터 기반 표시. 서버가 양쪽 인벤토리 관리 → 클라이언트는 표시만.
소모 시 시각적 제거 + 오브젝트 삭제, 재획득 시 다시 표시.

### ✅ Q16. 아이템 슬롯 UI 레이아웃 (2026-07-17)

1. 랜덤 슬롯은 **4×2 형태**로 배치.
2. **빈 랜덤 슬롯은 처음부터 보임** — 추가 저장 가능함을 플레이어에게 노출.

### ✅ Q17. 환경 시스템

데모에는 미포함. 이후 추가.

### ✅ Q18. 스마트폰 회복량 3→5→7° (2026-07-17)

**특수 기능으로 3회 사용 가능** (1회용 아님). 구현 난이도 문제 시 프로그래머와 회의 후 조치.
> ⚠ 코드 영향: HealPerUse=[3,5,7] 이미 지원 — **MaxUses=3 세팅 확인 필요**

### ✅ Q19. 안아줘요 티셔츠 역효과 (2026-07-17)

**의도된 리스크 (심리전 요소).** 내 온도가 더 높으면 상대를 회복시킴. (현재 코드와 일치 — 양방향 동기화)

### ✅ Q20. 라운드 간 리셋 범위 (2026-07-17)

| 항목 | 처리 |
|------|------|
| 온도 | 초기화 (37°) |
| 기본 영구 아이템 (부채, 바람막이) | 유지. 단, **강화 수치는 기본으로 초기화** (예: 선풍기 강화) |
| 기본 소모 아이템 (차, 고양이) | 초기화 (리필) |
| 랜덤 아이템 | 초기화 — **랜덤 4개 다시 지급** |
| 버프/디버프 (지연 효과 포함) | 초기화 |
| 온도 구간 지급 이력 (30/20/10°) | 초기화 (구간 칸 재생성) |

> ⚠ 코드 영향: 라운드 리셋 시 **FanSpeed 등 강화 수치 초기화 추가 필요**

### ✅ Q21. 버프/디버프 중첩 (2026-07-17)

> ⚠ 코드 영향: FanSpeed 강화 중첩 방지 플래그 추가 필요
